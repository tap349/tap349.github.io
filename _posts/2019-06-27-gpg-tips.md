encrypt/decrypt file with a symmetric cipher
--------------------------------------------

1. <https://www.gnupg.org/gph/en/manual/x110.html>

### use passphrase file

```gpg
# ~/.gnupg/gpg.conf

passphrase-file /Users/tap/.gnupg/passphrase
```

### encrypt

```sh
$ gpg -c foo.txt
$ ls
foo.txt foo.txt.gpg
```

### decrypt

```sh
$ gpg -d -o bar.txt foo.txt.gpg
$ ls
bar.txt foo.txt foo.txt.gpg
```
