(how to) get query SQL
----------------------

1. <https://hexdocs.pm/ecto/Ecto.Adapters.SQL.html#to_sql/3>

```elixir
> import Ecto.Query
> query = from p in User
#Ecto.Query<from u in User>
> Ecto.Adapters.SQL.to_sql(:all, Repo, query)
{"SELECT u0.\"id\", u0.\"name\" FROM \"users\" AS u0", []}
```
