## flush redis database

in rails console:

```ruby
> Sidekiq.redis { |redis| redis.keys.each { |key| redis.del key } }
```

this will print all deleted keys. to flush all keys silently:

```ruby
> Sidekiq.redis { |redis| redis.flushall }
```

or else do the latter in terminal:

```sh
$ redis-cli flushall
```
